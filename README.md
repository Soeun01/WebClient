# 우송대 도서관 메타버스
우송대학교 도서관을 온라인에서도 만날 수 있는 사이트
<br><br>

## 1. 주요 기능
- 웹 홈페이지
  - 도서관 메타버스 로그인/회원가입
  - 도서 검색
  - E-Book 대출/반납/찜
- 메타버스
  - 멀티 플레이
  - 채팅/귓속말
  - 회의룸
    - 캠
    - 마이크
    - 화면공유
<br><br>
